Exercise 2: Audio and Video Formats

In this exercise, you will make sure your audio and video flies work across different browsers. You will convert a video and audio into multiple formats so a browser can fallback on other file types if it does not support the original file type.

  • Download this file. Or find another .mov file you like.
  • Use Online Video Converter to encode (aka change the file into multiple formats) so that your video will play in multiple broswers like Firefox, Chrome, Sarafri, Opera, Internet Explorer.
  • Download & Save the encoded files into your project folder.
  • Create an video tag in your HTML to embed the video. Use multiple source codes to specify the type of file.
  • Make sure it plays on each browser.
  • Repeat the steps above using an audio file.
  • Bonus:Download JWPlayer and put the player.swf in your project folder. Add a flash fallback to your video tag. Specify the src URL like "player.swf?videoUrl=sayHello.mp3&autoPlay=true"
  • Bonus:Try recording a video on your smartphone and figuring out how to get that encoded and on your webpage.
